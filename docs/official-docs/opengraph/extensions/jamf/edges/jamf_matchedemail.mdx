## Edge Schema

Traversable: true

## General Information

The traversable `jamf_MatchedEmail` edge represents an identity correlation where the Jamf computer user's email
attribute matches the Jamf account's email, indicating they are likely the same person. This links physical device
access to Jamf administrative privileges.
