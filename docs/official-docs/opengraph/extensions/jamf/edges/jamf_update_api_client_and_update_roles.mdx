## Edge Schema

Traversable: false

## General Information

The non-traversable `jamf_Update_API_Client_and_Update_Roles` edge represents a combined permission where the source can
update existing API clients and update existing API roles. This is non-traversable because Jamf accounts and groups
cannot retrieve API client credentials without the 'Create API Integration' permission.
