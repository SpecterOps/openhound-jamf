## Edge Schema

Traversable: true

## General Information

The traversable `jamf_Okta_Same_Device` edge represents a hybrid cross-platform device correlation where the Jamf Pro
registered computer's UDID matches the registered device UDID in Okta. This optional edge is only created when the
`--okta` flag is enabled and links the Jamf device graph to OktaHound's device graph.
