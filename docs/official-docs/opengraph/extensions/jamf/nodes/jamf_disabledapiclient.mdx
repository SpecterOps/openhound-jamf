## Description

Represents a disabled Jamf Pro API client integration. Disabled API clients retain their role assignments but cannot
authenticate. If re-enabled, they regain all assigned permissions.

> This node shares its property set with [jamf_ApiClient](/opengraph/extensions/jamf/nodes/jamf_apiclient). The difference is that the client's
`enabled` property is false.
