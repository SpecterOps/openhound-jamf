## Description

Represents a disabled Jamf Pro account. Disabled accounts retain their permission configuration but cannot actively authenticate. If re-enabled, they regain all assigned privileges.

> This node shares its property set with [jamf_Account](/opengraph/extensions/jamf/nodes/jamf_account). The difference is that the account's `enabled`
> property is set to "Disabled".
